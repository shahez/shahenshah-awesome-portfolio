# Mohammad Shahenshah - Portfolio

Personal portfolio website showcasing my journey as a Staff Software Engineer with 10+ years of experience building scalable, AI-powered applications in healthcare, mobility, and enterprise systems.

## 🎯 About

Full-stack engineer specializing in:
- **AI-Powered Healthcare** - Clinical decision support at RapidAI
- **Cloud-Native Architecture** - AWS, Kubernetes, Docker, Microservices
- **Enterprise Systems** - Serverless architectures, IoT integration, CI/CD

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Three.js (3D animations)
- **Build Tool:** Vite
- **Styling:** CSS3 with custom animations
- **3D Graphics:** React Three Fiber, Drei
- **Physics:** Rapier (3D physics engine)
- **Animations:** GSAP with ScrollTrigger

## ✨ Features

- Interactive 3D tech stack visualization
- Smooth scroll animations
- Responsive design
- Career timeline showcase
- Project carousel with real company work
- Custom cursor effects
